Mr. Video Downloader

A powerful, yet simple Graphical User Interface for `yt-dlp`, designed for fast and efficient video downloads from various platforms.

Intuitive Interface

Why Choose Mr. Video Downloader?

Standalone & Portable

Download a single `.exe` file and run it immediately. No installation, no Python, no extra dependencies needed. `yt-dlp` is bundled inside!

Effortless Quality Selection

Simply paste your video URL, click 'Analyze', and choose from a comprehensive list of available video and audio formats and qualities.

Powered by `yt-dlp`

Leverages the industry-standard `yt-dlp` backend for robust, reliable, and up-to-date downloading capabilities from a vast array of sites.

Real-time Progress & Status

Stay informed with a clear progress bar and real-time status updates throughout your download process.

Custom Download Location

Easily browse and select any folder on your computer to save your downloaded media.

Free to Use (Private)

Download and use Mr. Video Downloader for your personal, non-commercial needs without any cost or subscriptions.

How to Get Started

1. Download the Executable

Grab the latest standalone `.exe` file directly from our GitHub Releases page. No installation requiredâ€”just download and run!

Download Mr. Video Downloader

2. Run and Paste URL

Simply double-click the downloaded `.exe` file. In the application, paste your desired video URL into the provided input box.

# Example URL
https://www.youtube.com/watch?v=dQw4w9WgXcQ

3. Analyze and Select Quality

Click the Analyze button. The application will fetch all available video and audio formats. Select your preferred quality from the dropdown menu.

4. Start Download

Confirm your save location (a `downloads` folder will be created by default) and click Start Download. Monitor the progress bar and status messages.
